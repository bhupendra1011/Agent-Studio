Knowledage Base
rules:

KB only have 3 status: Processing | Active | Partial_Failure which decided by:
If any docs under KB are still Uploading, Processing, then KB is Processing
If every docs are not Uploading. Processing,

then if any files are Upload_failed, Processing_Failed, then KB is Partial_Failure.
then if every files are Processed, then KB is Active

For docs, if doc is Upload_Failed, then there's no way to recover it. If doc is Process_Failed first time, it has a chance to process again, and if processing succeed, then doc would be update to Processed , otherwise the status doesn't change. (The doc processing final status is decided by RAG service)
